The Instant Social Network

CommunityEngine is a free, open-source social network plugin for Ruby on Rails applications. Drop it into your new or existing application, and you’ll instantly have all the features of a basic community site.

Download it now and get started

Features

  • Authentication (sign up, log in)
  • User profiles
  • User search
  • Blogs with tagging, categories and rich text editing
  • Photo uploading and tagging
  • “Clippings” - visual bookmarks
  • Commenting and comment notification by e-mail (Blogs, profiles, photos, and clippings can be commented)
  • Forums
  • Friendships and activity feeds
  • Favoriting of clippings and blog posts
  • Private messaging
  • And more!

The CommunityEngine plugin is very flexible, making it easy to customize every detail of your community’s appearance.

Real World Usage

The CommunityEngine plugin has been in use for over a year on www.curbly.com and www.uncooped.com, serving several hundred thousand visitors monthly.

Why are you releasing this?

Because I want to make it better. This plugin is extracted from code I’ve used on other projects, and I figured other people would benefit from using it. Still, there’s a lot of work yet to be done, and I’m hoping people decide to contribute and help improve the plugin.
