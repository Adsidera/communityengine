This plugin allows you to add Twitter integration to your CE app. Allow your users to sign up and log in using their Twitter credentials. Let them post status updates in the CE app, and have those automatically pushed to their Twitter streams. Import Tweets from Twitter into your users' CE app profiles.

Requires CommunityEngine version 1.0.4 or greater

Features

  • Easy installation
  • Increase sign ups by offering an additional, easy sign in option
  • Allow all users (not only those signed in through Twitter) to post status updates
